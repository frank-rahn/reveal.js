Einführung in Kotlin

Details: Beispiel Java<->Kotlin, Grundlegendes, Multiplatform, Entwicklungsgeschichte

Ein minimales Java-Programm


            package de.rahn.java.helloworld;

            public class Application {
              public static void main(String[] args) {
                System.out.println("Hello World!");
              }
            }

Kotlin!


            fun main(args: Array<String>): Unit {
              println("Hello World!");
            }

Kürzer


            fun main(args: Array<String>) {
              println("Hello World!")
            }

Einfacher


            fun main() {
              println("Hello World!")
            }

Minimalistisch


            fun main() = println("Hello World!")

Angemessenheit: Die Wahl der Komprimierung sollte dem Problem bzw. Aufgabe angemessen sein.

Grundlegendes zu Kotlin

  • Drastische Reduktion von Boilerplate Code
    • Typinferenz: der Typ kann oft weg-
      gelassen werden
    • Als Anweisungsende genügt häufig
      der Zeilenumbuch
  • Erzwungene Null-Sicherheit (Null Safety)
  • Freundlich zu Entwicklungswerkzeugen
Beispiel von Microsoft und dem SQL-Gremium (ISO/IEC) erzählen

Grundlegendes zu Kotlin

    • Kotlin kann in existierenden Projekte

  • Programmierung

    • Prozedural
    • Funktional
  • DSLs mit Kotlin
    • Beispiele: Gradle, Spring Framework
  • Objektorientiert
  • Prozedural: Prozeduren, ...

              println(name?.length)

if Anweisung

Datei: ControlFlow.kt

            package de.rahn.kotlin.lang

            fun max(a: Int, b: Int): Int {
              var max: Int
              if (a > b) {
                max = a
              } else {
                max = b
              }
              return max
            }

            fun controlFlow() {
              println("max(4, 5) = ${max(4, 5)}")
            }

            fun main() = controlFlow()
